# THD-PLV2 - Turtlebot3 maze challenges

During the week of the 7th of march in 2022, we mastered the [Turtlebot3 maze challenges](https://mygit.th-deg.de/gaydos/tb3-maze-challenges) in the curse KI-BI-PLV2 held by [Prof. Gökçe Aydos](https://aydos.de/uni).
For more information about the used technologies, please see the homepages of [Turtlebot3](https://www.turtlebot.com/) and [ROS - Robot Operating System](https://www.ros.org/). 
On this site, you can find the documentation by both team members.
